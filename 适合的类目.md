 产品核心逻辑

  你的产品是通过 网络搜索 + AI 分析 来生成预测报告：
  1. 从 Polymarket/Kalshi 获取市场问题
  2. 用 Valyu API 搜索相关证据
  3. AI 对证据进行权重分析
  4. 汇总生成概率和报告

  ---
  类目适合度分析

  | 类目          | 适合度     | 原因                                                 |
  |---------------|------------|------------------------------------------------------|
  | 政治/选举     | ⭐⭐⭐⭐⭐ | 新闻报道多、民调数据可验证、历史先例丰富、时间线明确 |
  | 地缘政治      | ⭐⭐⭐⭐⭐ | 官方声明多、专家分析丰富、事件有迹可循               |
  | 体育赛事      | ⭐⭐⭐⭐   | 统计数据完整、历史对战记录多、但伤病等实时信息难捕捉 |
  | 宏观经济指标  | ⭐⭐⭐⭐   | 官方数据、专业分析多，但涉及复杂模型                 |
  | 科技/产品发布 | ⭐⭐⭐     | 有泄露、分析师预测，但内部信息不透明                 |
  | 娱乐/奖项     | ⭐⭐⭐     | 有历史规律（如奥斯卡），但主观性强                   |
  | 加密货币新币  | ⭐⭐       | 信息碎片化、噪音多、缺乏可靠基本面                   |
  | 短期价格预测  | ⭐         | 市场随机性高、难以从公开信息获得优势                 |

  ---
  为什么加密新币分析效果差

  从你分享的报告可以看出问题：

  1. 证据都是通用的 - "市场热情高"、"社区活跃"这些是套话，没有具体到这个项目
  2. 缺乏可验证数据 - 新币没有历史、没有财务数据、没有可量化指标
  3. 信息来源噪音大 - 加密领域充斥着营销文、付费软文、虚假社区活动
  4. 时效性问题 - 市场变化太快，搜索到的信息可能已过时

  ---
  最适合的场景

  政治选举类 是最佳场景，因为：
  - 📊 有大量权威民调数据
  - 📰 主流媒体持续报道
  - 📅 有明确的决策时间点
  - 🔍 候选人立场、历史投票记录可查
  - 📈 有历史选举数据作为基准

  例如："共和党总统候选人提名"、"某州选举结果"这类问题，AI 能找到高质量、可验证的证据来支撑分析。

  ---
  简单说：你的产品适合 信息公开透明、有历史数据、有权威来源 的预测市场。不适合 高噪音、纯投机、缺乏基本面 的市场。
